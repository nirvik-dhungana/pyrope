## 🎨 Design Principles

Every project I build follows these core principles:

✨ **Intentional Design** — No defaults. Every color, spacing, and interaction is considered.

♿ **Accessibility First** — WCAG AA minimum. Keyboard navigation by default. Color-blind friendly.

⚡ **Performance** — Optimize for real users, real devices. Measure, iterate, ship fast.

🔤 **Semantic HTML** — Proper markup matters. Good markup = better accessibility + SEO.

🎯 **User-Centric** — Design for the actual person using the interface, not the designer.
